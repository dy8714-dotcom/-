# 🍅 ポモドーロタイマー

シンプルで使いやすいポモドーロテクニックタイマーです。PC・スマホ・タブレットで動作します。

## ✨ 主な機能

### ⏱️ カスタマイズ可能
- 作業時間・休憩時間を1〜99分まで自由に設定
- 複数ターン(1〜20回)の繰り返し対応
- デフォルト: 作業25分 / 休憩5分 / 4ターン

### 🔔 柔軟な通知機能
- **作業中の通知**: 4つのタイミングで通知可能(例: 残り15分、10分、3分、1分)
- **休憩中の通知**: 必須の休憩通知(例: 残り休憩1分です)
- **整時刻通知**: 毎正時(11:00、14:00など)に控えめなチャイム音

### 🎙️ 音声アナウンス
各通知で音声タイプを選択可能:
- 🔔 音のみ
- 🎙️ 音声アナウンス(女性の声)
- 🔔🎙️ 音+音声アナウンス

終了時も選択可能:
- 🔔 アラーム音のみ
- 🎙️ 「お疲れ様でした」音声のみ
- 🔔🎙️ 音+音声
- 🔇 ミュート(文字のみ)

### 📱 レスポンシブデザイン
- PC、スマホ、タブレットに自動対応
- 縦向き・横向きどちらでも快適に使用可能
- ホーム画面に追加してアプリのように使える

## 🚀 使い方

### Webで使う(おすすめ)
GitHub Pagesで公開されています:
```
https://あなたのユーザー名.github.io/pomodoro-timer/
```

### ローカルで使う
1. `index.html`をダウンロード
2. ブラウザで開く
3. すぐに使えます!

## 📱 スマホでの使い方

1. ブラウザでURLを開く
2. メニューから「ホーム画面に追加」を選択
3. アイコンをタップして起動!

### iPhone/iPad
Safari > 共有ボタン > ホーム画面に追加

### Android
Chrome > メニュー(⋮) > ホーム画面に追加

## ⚙️ 設定例

### 集中モード
- 作業: 25分
- 休憩: 5分
- ターン: 4回
- 通知: 残り3分、1分のみ

### 長時間作業モード
- 作業: 50分
- 休憩: 10分
- ターン: 3回
- 通知: 残り15分、10分、3分、1分

## 🎨 デザイン

- 黒背景にネオングリーンの数字で視認性抜群
- シンプルで邪魔にならないデザイン
- 直感的な操作性
